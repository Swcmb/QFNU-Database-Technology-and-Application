## 实验三扩展：使用 WALExplorer 工具分析 WAL

 项目地址  https://github.com/Swcmb/WALExplorer 

### 1.1 导出 WAL 文件为文本格式

```bash
# 导出 WAL 文件为文本格式
pg_waldump /home/uxdb/uxdb/pg_wal/000000010000000000000001 > wal_physical.txt
```

### 1.2 使用 WALExplorer 解析 WAL

```bash
# 进入 WALExplorer 目录
cd /root/db/WALExplorer

# 解析文本格式的 WAL 文件
python main.py ./wal_physical.txt > res.txt
```

### 1.3 查看解析结果

```bash
cat res.txt
```

WALExplorer 会将 WAL 记录转换为可执行的 SQL 语句，方便你理解 WAL 中记录的操作。
